game: 杰克兄弟
file: Jack Bros. (USA).vb
developer: 杰克兄弟
description: 1995年的冒险游戏，你需要从杰克三兄弟中选择一人进行操控，每个人都性能各异，有自己的专属武器和特殊攻击。本作部分角色来自于女神转生系列，而且也是地牢性质，不过玩法不同；攻关方法是每层找到全部钥匙然后不断向下跳打倒Boss。

game: 马里奥冲击
file: Mario Clash (Japan, USA).vb
developer: 马里奥冲击
description: GB之父横井军平亲自带领研发一组制作的益智类游戏，是经典街机版《马里奥兄弟》的升级版，只不过这次你需要扔龟壳打倒敌人（所以有点更像家用机的马里奥了）。游戏相当有趣，而且量大管饱（一共99关！）。

game: 马里奥网球
file: Mario's Tennis (Japan, USA).vb
developer: 马里奥网球
description: 这是每台VirtualBoy系统都自带的一款游戏。游戏没太大特点，就是用马里奥系列的角色打网球罢了。背景的层次和角色视角上比同时期GB版本要好一点，但是没有N64的好。其实如果要达到最佳的VR效果，游戏加一个可选择的第一人称视角肯定会更好一些，可惜这只是一个假设。

game: 奈斯特的时尚保龄球
file: Nester's Funky Bowling (USA).vb
developer: 奈斯特的时尚保龄球
description: 和很多VB游戏一个问题，就是这款游戏完全可以出在其他平台上，没有太用到VB机能的特殊之处。值得一提的是，它是以老游戏杂志《任天堂力量》上连载漫画人物奈斯特为主角做的游戏，而且角色的细节还是比较生动的。

game: SD高达-维度战争
file: SD Gundam - Dimension War (Japan).vb
developer: SD高达-维度战争
description: VB上最后一款作品，也是高达游戏史上较为冷门的作品。游戏形式是常规的SRPG，有特色的地方在于特写战斗时的视角和立体感，比较新鲜。

game: 遥控机器人拳击手
file: Teleroboxer (Japan, USA).vb
developer: 遥控机器人拳击手
description: 第一人称的拳击游戏，可谓是平台上少有的能体现VB优势的精品。通过两个方向键格挡，通过扳机键出拳。这款游戏体现了VB的潜力所在，不过可能是由于VB前期的保密性，导致几乎没有其他开发商能做到这一点。

game: 垂直战机
file: Vertical Force (USA).vb
developer: 垂直战机
description: 飞行射击游戏，利用到了VB产生高低差视觉的机能。游戏的特色是飞机机体可以上下移动。

game: 瓦里奥大陆VB
file: Virtual Boy Wario Land (Japan, USA).vb
developer: 瓦里奥大陆VB
description: VirtualBoy平台上公认的最好玩的游戏，只能说是错生在了这个任天堂历史销售最差（80万）的游戏机上了。游戏的质量很高，可以和GB/GBC上的瓦里奥大陆一较高下，玩法基本相同，但还有许多利用到VB特性的设计，有3D效果，强烈推荐！
